# Proudct-Management
Express+MongoDb 商品管理系统

该系统利用node.js的Express框架以及MongoDb作为数据库构建的一个网站，该网站目前只有后台管理员登录、退出登录；后台登录权限管理；后台商品列表，增加商品，修改商品，删除商品；后台商品图片上传；MongoD数据库封装。该项目为练手项目，后续更新其余商城系统应有的功能.......
